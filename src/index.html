Instrucciones:

Existen 2 endpoints requeridos por consumir:

1.GET http://server:55000/obtener_horarios :
Devuelve los horarios de los docentes, recibe los siguientes parametros:
{
nombre_docente: nombre (es opcional, si no se le provee, trae los de todos),
fecha_clase: fecha
}

2.GET http://server:55000/obtener_docentes: Devuelve los docentes existentes

Utilizar la fecha de consulta 2018-04-06 como su fecha de clases actual, para todas las consultas
